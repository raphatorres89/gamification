# Gamification

Este sistema de gamificação foi criado para incentivar os alunos a aprenderem de forma mais divertida e interativa. O objetivo é responder as perguntas com o intuito de derrotar um boss, cada resposta correta concede experiência e causa um dano ao boss, que quando chega a zero de vida, é derrotado. No final um ranking é exibido com os vencedores.

## Como usar
1. Clone o repositório para sua máquina local.
2. Abra o arquivo `index.html` em um navegador web (locahost:8080)

## Tecnologias utilizadas
- HTML
- CSS
- JavaScript
- Java (para o backend)
- Spring Boot (para o backend)
- H2 (para o banco de dados)
- Gradle (para gerenciamento de dependências)
